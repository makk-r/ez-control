### HOW TO INSTALL

1.แตกไฟล์ตามขั้นตอนของแต่ละos

2.ติดตั้งโมดูล (ต้องมี python ก่อน)

~~~bash
pip install -r ez-control/requirements.txt
~~~

3.เปิดโปรแกรม

~~~bash
python ez-control
# or
python3 ez-control
~~~
