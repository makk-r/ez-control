# รายละเอียดการอัพเดท 26.2a1 (ไม่ได้บันทึกวันที่)

การเปลี่ยนแปลงระบบ
1. การเพิ่ม การตรวจเช็คอัพเดทระบบด้วยคำสั่ง update get

2. การเพิ่ม การเชื่อมต่อ github

3. การเพิ่ม ระบบเซฟ repo github

การแก้ไข
1. การเปลี่ยนแปลง 
2. การเปลี่ยนคำสั่งเพื่อให้เรียกง่ายขึ้น

มีหลายอย่างที่มีการอัพเดทแต่หลักๆมีประมาณนี้

และจะปล่อยเวอร์ชั่นเต็มเวอร์ชั่น 26.2 อีกไม่นาน

# รายละเอียดการปล่อย 26.2 (ไม่ได้บันทึกวันที่)

การแก้ไข
1. ลดปัญหาโค้ดซ้ำซึ่งทำให้เกิดการกินพื้นที่

มีหลายอย่างที่มีการอัพเดทแต่หลักๆมีประมาณนี้
เราทำการตรวจเช็คระบบมากกว่า 85 เปอร์เซ็นพร้อมทำงาน
และฟังก์ชันต่างๆในเวอร์ชั่น Alpha ไม่ได้มีการลบทิ้งไปไหน
